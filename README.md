# InitialProposal_Pac-Man

This is a proof of concept for my 3296 Software Design project proposal. A simple implementation of Pac-Man (with no enemies yet) using Pygame.

## My system
This program was able to compile on a:
- **Operating System**: MacOS
- **Python version**: 3.12.1
- **Pygame version**: 2.6.1

This program should be able to run on Windows, Mac, and Linux operating systems. Anything that can install Python and Pygame.

## How to run and compile 
In the terminal:
1. git clone [https://github.com/dylanvo1203/InitialProposal_Pac-Man]
2. cd [cloned repository]
3. python pacman.py

## How to play
- **Arrow Keys**: Move Pac-Man
- **Close Window**: Quit the game

## Gameplay
- Collect all white pellets
- Each pellet is worth 10
- Win by collecting all pellets
